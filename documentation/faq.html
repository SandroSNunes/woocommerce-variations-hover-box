Is this plugin suited for shops that has variations products with only 1 attribute?

Yes. For more attributes, please buy the premium plugin.

Can I use the plugin with on a shop that has variation products with multiple attributes, like size and color?

You can, but on the shop page, when you click on the attribute button, it won't add the product automatically to the cart, but instead it will open the product page, so that you can select all the attributes.
